# Hermes Dashboard

Веб-интерфейс для AI-агента **OWL** — персональный дашборд в стиле премиум AI-продуктов (Linear / Raycast / Arc).

## Стек

- React 18 + Vite + TypeScript
- Tailwind CSS v4 (CSS-first `@theme` конфигурация)
- Framer Motion для анимаций
- Zustand (с persist) для состояния
- React Router для навигации
- lucide-react для иконок

## Возможности

- Вкладки сессий с drag & drop, tooltip и анимированным появлением
- Чат в стиле iMessage с markdown-рендерингом и подсветкой блоков кода
- Карточки вызовов инструментов с прогресс-баром и индикацией статуса
- Анимированный OWL-аватар (моргание, "думает", "радуется", "грустит")
- Статус-бар с моделью `openrouter/owl-alpha`
- Сворачиваемый сайдбар с поиском и группировкой (сегодня/вчера/раньше)
- Панель настроек: тема, акцент, размер шрифта, API endpoint, горячие клавиши
- Горячие клавиши: `⌘N`, `⌘W`, `⌘K`, `⌘/`, `Esc`
- Приветственный экран с быстрыми действиями
- Демо-сессии при первом запуске

## Запуск

```bash
pnpm install
pnpm dev
```

Откройте http://localhost:5173/.
